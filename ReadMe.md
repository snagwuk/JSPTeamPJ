# 웹페이지 소개 및 기능
-  가상의 체육문화센터 웹페이지
- 마이페이지 : 내정보수정,  나의수강목록,나의 대관신청목록, 나의민원목록
- 관리자 페이지 :  미답변 민원 / 대관 신청 리스트 , 대관신청 승인/거절 , 민원 답변 , 
                  회원목록관리, 대관장소관리,강사관리, 대관신청목록
 - 수강신청 - 프로그램 안내, 수강신청, 강의평가
 

 - 대관신청 - 대관장소안내, 대관신청
 

 - 고객의소리 - 민원게시판(좋아요기능)
 

 - 오시는길 - KAKAO 지도 API 이용
 
 - 프로그램 수강현황 - CHART 그래프 (chart.js)
